Algoritmo Regressão linear

A análise de regressão é uma técnica estatística que investiga e modela relações que variáveis possam ter entre si. As aplicações da regressão ocorrem em quase todos os campos como a engenharia, a economia, a gestão entre várias outras áreas de conhecimento. Sendo um conceito amplo, a análise de regressão possui várias técnicas, entre elas está a técnica de regressão linear. A técnica de regressão linear busca prever o valor de dados desconhecidos usando como base valores de dados relacionados e conhecidos. Ela se baseia na ideia de que existe uma relação linear entre uma variável dependente e uma ou mais variáveis independentes. Para entendermos o problema de regressão linear, vamos observar a situação proposta [1]: Suponha que um engenheiro industrial empregado por uma engarrafadora de bebidas analise as operações de entrega de produtos e serviços para máquinas de venda. Ele suspeita que o tempo necessário para um entregador de rota carregar e atender a uma máquina está relacionado ao número de caixas de produtos entregues. O engenheiro visita 25 pontos de venda escolhidos aleatoriamente que possuem máquinas de venda. O tempo de entrega (em minutos) e o volume de produtos entregues (em caixas) são observados para cada uma das máquinas de venda.


No gráfico acima estão representadas as 25 visitas que o engenheiro realizou. A representação sugere claramente uma relação entre o tempo de entrega e o volume de entrega, ilustrando a relação linear do problema. Se representarmos y como o tempo de entrega e x como o volume de entrega, a equação de uma linha reta que relaciona essas duas variáveis é y = β0 + β1.x, onde β0 é a interceptação e β1 é a inclinação. Agora, os pontos de dados não se ajustam exatamente a uma linha reta, portanto, a equação deve ser modificada para levar isso em consideração. Esse ajuste que aplicamos sobre a equação trata-se da diferença entre o valor observado de y e a reta (β0 + β1.x), na qual chamamos de erro ε. É conveniente pensar em ε como um erro estatístico; ou seja, é uma variável aleatória que considera a falta de ajuste exato do modelo aos dados. O erro pode ser composto pelos efeitos de outras variáveis no tempo de entrega, erros de medição e assim por diante. Portanto, um modelo mais plausível para os dados de tempo de entrega é y = β0 + β1.x + ε. Esse tipo de regressão é uma linha de valores médios, isto é, representa um valor médio que esperamos para y quando x tem um valor qualquer atribuído. No modelo, x é chamado de variável independente e y é chamado de variável dependente. β1 é o valor que indica a inclinação da reta, podendo ser interpretada como a mudança na média de y para uma mudança de uma unidade em x. β0 é a constante que determina a posição vertical da reta. Ele representa o valor esperado de y quando x é igual a zero.
